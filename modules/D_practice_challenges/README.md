# Module D – Practice Challenges

Standalone tasks that mirror common Coderbyte DevOps questions.

Included:

- `log_parser/` – parse errors and generate JSON summary
- `api_client/` – call HTTP API and filter data
- `dockerfile_fix/` – fix and optimize Dockerfile
- `terraform_bucket/` – simple storage resource with variables
- `k8s_deployment/` – fix broken Deployment

Each folder has a `task.md` description and starter or solution file.
