# Google Cloud - From Zero to Hero

This module is a comprehensive, hands-on guide to mastering Google Cloud Platform (GCP). It is designed to take you from the fundamentals of cloud computing on GCP to advanced topics, preparing you for real-world DevOps, SRE, and Cloud Architect roles.

## Structure

The curriculum is divided into modules, each focusing on a core area of GCP. Every module includes:
- **README.md**: In-depth explanations of key concepts and services.
- **Labs**: Hands-on problems and tasks to build practical skills.
- **Code Examples**: Sample code in Terraform, Bash, and Python to demonstrate implementation.

### Learning Path

1.  **01-gcp-foundations**: Core concepts - Projects, Billing, IAM, and the gcloud CLI.
2.  **02-compute**: Compute Engine, instance management, and scaling.
3.  **03-storage-databases**: Cloud Storage, Cloud SQL, Spanner, and Firestore.
4.  **04-networking**: VPCs, subnets, firewall rules, and load balancing.
5.  **05-containers-gke**: Google Kubernetes Engine (GKE), containerization, and orchestration.
6.  **06-serverless**: Cloud Functions and Cloud Run.
7.  **07-devops-sre**: Cloud Build for CI/CD, Cloud Monitoring, and Logging.
8.  **08-security**: Cloud Armor, Security Command Center, and service account best practices.
9.  **09-big-data-ml**: Introduction to BigQuery and Vertex AI.
